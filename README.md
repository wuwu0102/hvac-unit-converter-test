# HVAC 空調機房工程工具

本工具不是單純單位換算器，而是整合：
- HVAC 基礎換算
- 水管管徑與壓差流量估算
- 換氣量計算
- 冷負載估算
- 風量估算
- 機房負載 / 散熱 / 耗電 / 380V 三相電流概算

## 特色
- 保留正式版既有工具入口：溫度、流量、壓力、流速、電力換算、水系統估算、電力估算、意見回饋
- 新增中文化工程計算：換氣量、冷負載、風量、資料中心機房負載與耗電概算
- 支援手機與桌機版面
- 全部計算在本地端執行，不需登入、不使用外部 API

## 壓差估算流量工具

- 依設備參考點估算目前流量
- 管徑改為標準選單，降低輸入錯誤
- 自動估算流速
- 超過 3 m/s 時提醒放大管徑或重新確認設計條件
- 管徑內徑為工程估算值，實際需依管材與標準確認
