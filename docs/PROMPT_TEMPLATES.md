# Prompt Templates for AI Development

## Safe Feature Update Prompt

Please update only the necessary files for this feature.
Do not rewrite the whole app.
Preserve all existing routes, pages, and calculator functions.
Keep formulas in reusable utility modules.
Prevent NaN, Infinity, and undefined outputs.
Run build and summarize changed files.

## Engineering Calculator Prompt

Add or update this engineering calculator using:
- controlled inputs where possible
- explicit units
- formula comments
- estimated labels where applicable
- mobile-first layout
- defensive validation
- no hidden constants in UI components

## Bug Fix Prompt

Find the smallest fix for this bug.
Do not change unrelated behavior.
Add validation if needed.
Run build.
Explain the root cause and modified files.
